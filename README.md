# config.nvim

My neovim config.

## installation

If you have a pre-existing config at [~/.config/nvim](~/.config/nvim) rename it to something like `nvim.old`.

### dependencies

If using neovim v0.12.*, you will need to install tree-sitter-cli using your package manager of choice.

```bash
cd ~/.config && git clone git@github.com:anton-oz/config.nvim
```
